# ThinkTrace AI 🧠
> Watch AI Think. — Powered by Qubrid AI × Kimi K2 Thinking

## What it does
Ask any complex question — math, logic, science, coding — and watch Kimi K2 
Thinking's full reasoning process unfold in real time, step by step, before 
delivering a verified final answer.

The reasoning trace IS the product. This app makes the model's thinking 
process visible and explorable.

## Setup
1. Clone the repo
2. Copy .env.example → .env, add QUBRID_API_KEY
3. Run `uv sync` to install dependencies
4. Start the app with `uv run streamlit run app.py`

## What makes this different
Most AI apps hide the thinking. ThinkTrace AI surfaces it — every step, 
every assumption, every check — using Kimi K2 Thinking's native 
reasoning_content field via the Qubrid AI platform.
